Язык программирования: C#. Графическое приложение с использованием WinForms. 
Моделирование услуг автопарка, предоставляющего аренду автомобилей населению
Заказчик: оформление заказа, аренда транспорта. 
Менеджер: проверяет правильность заполнения, прилежность заказчика и возвращает ему ответ о его заказе, а также заполняет ведомость о проведенной аренде.
Директор: экономический отчет, контроль определенного сотрудника, пользуясь данными о сотрудниках.	

//

Для создания графического приложения для моделирования услуг автопарка, можно использовать язык программирования C# и библиотеку WinForms.

Первым шагом необходимо создать интерфейс пользователя, который будет включать в себя формы для оформления заказа и для аренды транспорта. Для этого можно использовать различные элементы управления WinForms, такие как кнопки, текстовые поля, выпадающие списки и т.д.

Затем следует реализовать логику приложения, которая будет обрабатывать данные, введенные пользователем в формы. Для этого можно использовать классы и методы C#, а также базы данных для хранения информации о клиентах и автомобилях.

Для работы с заказами можно создать класс Order, который будет содержать информацию о клиенте, автомобиле, дате заказа и сроке аренды. Для обработки заказов можно создать класс OrderManager, который будет проверять правильность заполнения заказа и возвращать ответ клиенту.

Для работы с арендой транспорта можно создать класс Rental, который будет содержать информацию о клиенте, автомобиле, дате аренды и сроке аренды. Для обработки аренды можно создать класс RentalManager, который будет проверять правильность заполнения заявки на аренду и заполнять ведомость о проведенной аренде.

Для работы с данными о сотрудниках можно создать класс Employee, который будет содержать информацию о сотруднике, его должности и другие сведения. Для обработки данных о сотрудниках можно создать класс EmployeeManager, который будет отвечать за экономический отчет и контроль определенного сотрудника.

Также необходимо создать базу данных для хранения информации о клиентах, автомобилях и сотрудниках, а также для сохранения информации о заказах и аренде транспорта. Для этого можно использовать технологию ADO.NET, которая позволяет работать с базами данных в C#.
